\# 📊 Power BI Sales Dashboard

\#\# 🔍 Project Overview    
This Power BI dashboard analyzes global sales data from 2015 to 2018\. It provides insights into total sales, top-performing regions and segments, product category performance, and shipping efficiency. The dashboard is fully interactive with slicers for region, segment, category, and more.

\#\# 📁 Files Included    
\- \`PowerBI\_SalesDashboard.pbix\` – Power BI dashboard file    
\- \`SalesDashboard\_Overview.pdf\` – Snapshot of full dashboard    
\- \`README.md\` – Project summary and usage instructions

\#\# 🧩 Dataset    
\- \*\*Source\*\*: Kaggle    
\- \*\*Dataset Name\*\*: Superstore Sales Dataset    
\- \*\*Link\*\*: \[https://www.kaggle.com/datasets/rohitsahoo/sales-forecasting\](https://www.kaggle.com/datasets/rohitsahoo/sales-forecasting)

\#\# 💡 Key Features    
\- KPIs: Total Sales, Average Sales, Order Count    
\- Sales performance by Year, Region, and Segment    
\- Top & Low Performing Product Categories    
\- Monthly sales trend chart    
\- Regional sales map    
\- Average shipping days by mode    
\- Interactive filters (region, category, sub-category, segment)

\#\# 🛠 Tools Used    
\- Power BI Desktop    
\- Microsoft Excel (for data preprocessing)    
\- Dataset from Kaggle

\#\# 🚀 How to Use    
1\. Download the \`.pbix\` file    
2\. Open it using Power BI Desktop    
3\. Interact with slicers to explore trends and breakdowns
